Shoefinder is a Chrome extension that helps users find the right shoe size when shopping online.

## Features
- Brand-based size recommendations
- Popup UI for user input
- Content scripts for product pages

## Tech Stack
- JavaScript
- HTML/CSS
- Chrome Extensions (Manifest V3)

## How to Run
1. Go to chrome://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select the project folder
